# Backend do UniVerse

## Configuração

### Variáveis de ambiente

1. Faça uma cópia do arquivo [`.env.example`][exemplo-env];
2. Renomeie a cópia para `.env`;
3. Preencha as variáveis contidas na cópia.

### Execução

1. `npm install`
2. `npm run dev:server`
